## Installation

You can install the package via composer:

```bash
composer require spatie/ray-bundle
```

## Usage

```php
$skeleton = new Spatie\RayBundle();
echo $skeleton->echoPhrase('Hello, Spatie!');
```

## Testing

```bash
composer test
```
